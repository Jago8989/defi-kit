# Actions

A set of standard actions.

#### `deposit`

Deposing and withdraing funds into a money market or AMM pool.

##### Parameters

| Name     | Type   | Description                                                          |
| :------- | :----- | :------------------------------------------------------------------- |
| `target` | string | The ID of the money market or AMM pool, ususally a contract address. |

#### `stake`

Stake and unstake LP or deposit tokens into a vault.

#### `harvest`

Harvest rewards from a vault.

#### `swap`

Swap tokens on a DEX.
